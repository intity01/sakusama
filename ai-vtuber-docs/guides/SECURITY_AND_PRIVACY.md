# Security & Privacy Guide

เอกสารนี้อธิบายแนวทางการออกแบบด้านความปลอดภัยและความเป็นส่วนตัวของ AI VTuber Desktop Companion

## ภาพรวม

เรายึดหลัก **Privacy-First** ในการออกแบบ ซึ่งหมายความว่าข้อมูลของผู้ใช้จะถูกปกป้องอย่างสูงสุด และผู้ใช้สามารถควบคุมข้อมูลของตนเองได้อย่างเต็มที่

## ส่วนประกอบหลัก

### Privacy Manager

เป็นศูนย์กลางในการจัดการนโยบายความเป็นส่วนตัวทั้งหมด ผู้ใช้สามารถตั้งค่าต่างๆ ได้ผ่าน Privacy Dashboard ใน UI

### Data Encryption

- **Encryption at Rest:** ข้อมูลที่ละเอียดอ่อนทั้งหมด (ประวัติการสนทนา, โปรไฟล์ Persona, API keys) จะถูกเข้ารหัสด้วย **AES-256** ก่อนบันทึกลงในดิสก์
- **Encryption in Transit:** การสื่อสารกับบริการภายนอก (เช่น OpenAI API, Cloud Sync) จะต้องผ่าน **TLS 1.3** เสมอ
- **Key Management:** Encryption keys จะถูกจัดเก็บอย่างปลอดภัยใน Key Store ของระบบปฏิบัติการ (เช่น Windows Credential Manager, macOS Keychain)

### Permission Controller

ระบบปลั๊กอินของเราทำงานใน **Sandbox** เพื่อจำกัดการเข้าถึงทรัพยากรของระบบ ปลั๊กอินจะต้องประกาศสิทธิ์ที่ต้องการในไฟล์ `plugin.json` และผู้ใช้จะต้องอนุมัติสิทธิ์เหล่านั้นก่อนปลั๊กอินจะทำงานได้

**ตัวอย่างสิทธิ์:**
- `read_messages`: อ่านประวัติการสนทนา
- `send_messages`: ส่งข้อความในนามของ AI
- `access_filesystem`: เข้าถึงไฟล์ในเครื่อง
- `access_network`: เชื่อมต่ออินเทอร์เน็ต

### Analytics Opt-in/out

เราอาจมีการเก็บข้อมูลการใช้งานเพื่อปรับปรุงโปรแกรม แต่ข้อมูลทั้งหมดจะถูก **anonymize** (ทำให้ไม่สามารถระบุตัวตนได้) และผู้ใช้สามารถเลือกที่จะ **ไม่ส่งข้อมูล** ได้ตลอดเวลาผ่าน Privacy Dashboard

### Data Audit Log

ทุกการกระทำที่เกี่ยวข้องกับข้อมูลส่วนตัว (การเข้าถึง, การสร้าง, การลบ) จะถูกบันทึกไว้ใน Audit Log ผู้ใช้สามารถตรวจสอบ log นี้ได้เพื่อดูว่ามีใครหรือโปรแกรมส่วนไหนเข้าถึงข้อมูลของตนบ้าง

## Best Practices

- **Local First:** ข้อมูลจะถูกเก็บไว้ในเครื่องของผู้ใช้เป็นหลัก การซิงค์กับคลาวด์เป็นเพียงทางเลือก
- **Principle of Least Privilege:** แต่ละส่วนของโปรแกรม (รวมถึงปลั๊กอิน) จะได้รับสิทธิ์เท่าที่จำเป็นในการทำงานเท่านั้น
- **Transparency:** ผู้ใช้สามารถเห็นได้เสมอว่าข้อมูลของตนถูกใช้อย่างไรและโดยใคร
- **Data Deletion:** ผู้ใช้มีสิทธิ์ที่จะลบข้อมูลทั้งหมดของตนเองได้อย่างง่ายดาย
