# AI VTuber Desktop Client

แอปพลิเคชันบนเดสก์ท็อปสำหรับ AI VTuber Desktop Companion ที่สร้างด้วย Electron

## โครงสร้างโปรเจกต์

```
ai-vtuber-desktop-client/
├── src/
│   ├── main/          # Electron Main Process
│   ├── renderer/      # Electron Renderer Process
│   ├── overlay/       # Transparent Overlay Window
│   └── ui/            # UI Components
├── public/            # Static assets
├── assets/            # Images, icons, etc.
├── tests/             # Tests
└── docs/              # Documentation
```

## คุณสมบัติหลัก

- **Transparent Overlay:** หน้าต่างโปร่งใสที่แสดงตัวละคร VTuber
- **Click-Through:** คลิกทะลุได้เพื่อไม่บดบังการทำงานอื่น
- **Multi-Platform:** รองรับ Windows, macOS, และ Linux
- **Customizable UI:** ปรับแต่ง UI ได้ตามต้องการ
- **System Tray:** ควบคุมผ่าน system tray
